功能特性 ​

蹭蹭车是一款专注于熟人社交的互助出行平台，提供安全、便捷、合规的拼车服务。

🎯 核心场景 ​

职场通勤 ​

- **企业圈子**: 员工内部拼车，增强同事交流 - **固定路线**: 每日通勤路线，时间地点固定 - **信用保障**: 实名认证 + 企业背书，安全放心

团建活动 ​

- **智能分配**: 自动匹配车主和乘客，优化路线 - **费用 AA**: 油费过路费自动计算，公平透明 - **活动管理**: 报名统计、车辆分配一键完成

邻里互助 ​

- **小区圈子**: 邻里拼车，互帮互助 - **免费互助**: 纯粹邻里情谊，不涉及费用 - **社区氛围**: 构建和谐邻里关系

💳 付费模式 ​

模式说明适用场景
免费互助完全免费，纯粹互助邻里拼车、同事互助
AA 分摊油费过路费 AA 制长途出行、自驾游
付费乘车平台限价，合规运营日常通勤、机场接送

🔐 安全机制 ​

信任体系 ​

  • ✅ 实名认证: 微信授权 + 手机号验证
  • ✅ 企业审核: 企业圈子需管理员审核
  • ✅ 双向评价: 司乘双方互评，信用积累
  • ✅ 信用分系统: 根据评价和履约行为计算

合规保障 ​

  • ✅ 限价机制: 付费模式设置价格上限
  • ✅ 行程记录: 完整行程轨迹可追溯
  • ✅ 评价审核: 恶意评价过滤和处理
  • ✅ 投诉通道: 纠纷处理和仲裁机制

📊 数据能力 ​

智能算法 ​

  • 车辆分配: 基于部门、位置、时间的多目标优化
  • 路线规划: 实时路况 + 历史数据，最优路线推荐
  • 供需预测: 基于历史数据的运力预测

运营分析 ​

  • 用户画像: 出行习惯、信用评估
  • 热力地图: 需求热点、线路分析
  • 运营报表: 订单量、转化率、留存率

🛠 技术优势 ​

高性能架构 ​

  • NestJS + TypeORM，模块化开发
  • Redis 缓存，毫秒级响应
  • MySQL 集群，支持百万级数据

小程序体验 ​

  • 原生微信小程序，即用即走
  • 流畅交互体验
  • 离线缓存支持

DevOps ​

  • Docker 容器化部署
  • 自动化测试和 CI/CD
  • 7x24 小时监控告警
